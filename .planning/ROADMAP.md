# Roadmap: postliterate-site

## Overview

Scaffold an Astro + MDX blog, apply the PostLiterate design spec, wire up RSS and publishing scripts. Three phases, straight through to a locally buildable site ready for GitHub + Vercel.

## Domain Expertise

None

## Phases

- [ ] **Phase 1: Foundation** — Astro project, MDX, content collection, base config
- [ ] **Phase 2: Pages & Design** — Layout, index page, post template, full design spec
- [ ] **Phase 3: RSS & Scripts** — RSS feed, publish.sh, preview.sh, sample post

## Phase Details

### Phase 1: Foundation
**Goal**: Working Astro project with MDX support and content collection schema
