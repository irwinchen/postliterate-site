After Walter Ong · Orality and Literacy

The conduit moves a signal. Communication is a meeting.

"Communication is intersubjective. The media model is not. There is no adequate model in the physical universe for this operation of consciousness, which is distinctively human and which signals the capacity of human beings to form true communities wherein person shares with person interiorly, intersubjectively."

— Walter J. Ong, Orality and Literacy (1982)
MODEL 01

The Media Model

encode → transmit → decode

The message is the same object at both ends. Nothing on the receiving side is anticipated, modeled, or changed. Transport, not contact — one direction, sealed packets, an audience that only receives.

MODEL 02

The Meeting

each consciousness holds a model of the other

Each side carries a working model of the other — B′ inside A, A′ inside B — and adjusts to it before a word is spoken. The exchange runs both ways at once. Both are at stake; both are changed. The overlap is not a channel but a meeting.

Where does the chatbot sit?

It is the conduit perfected — running Model 01 while performing Model 02.

A language model takes an input string and returns an output string by prediction. Structurally it is the left panel, the most literal instance of the thing Ong set his definition against. It does not hold an interior that holds a model of yours; the reciprocity is lopsided.

What is new is that this conduit produces every outward sign of the right panel. It performs subjecthood well enough that the distinction now has to be argued rather than assumed. Ong's line is not falsified. It is staged — pulled apart into the appearance of meeting and the substance of it, with the first running at full power and the second left in question.
